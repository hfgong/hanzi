# Hanzi 字典

一款支持漢字部件拆分的離線字典 Web App。

## 功能
- 同時支持簡體與繁體輸入，查詢結果會列出對應的轉換形式
- 顯示拼音、釋義、部件拆分
- 反向部件查詢

## 安裝與啟動

### 克隆項目

```bash
git clone <你的倉庫地址>
cd hanzi
```

## 數據說明

完整數據位於 `public/data/` 目錄，格式為預處理好的 JSON，可自行更新。

## 預處理腳本

`preprocess/prepare_data.py` 可用於整理與更新數據源。
